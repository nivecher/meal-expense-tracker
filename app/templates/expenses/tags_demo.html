Custom Tags Demo

This demo showcases the custom tags feature with Jira-style labels for organizing expenses.

Key Features

  • Dynamic Tag Creation: Create tags on-the-fly while adding expenses
  • Jira-Style Naming: Automatic normalization to lowercase with hyphens
  • Color-Coded Badges: Each tag gets a unique color for easy identification
  • Auto-Complete: Smart suggestions based on existing tags
  • Persistent Storage: Tags are saved and available across sessions
  • Multi-Select: Add multiple tags to a single expense
  • Search & Filter: Find expenses by tags
  • User-Specific: Each user has their own tag collection
  • Responsive Design: Works seamlessly on mobile and desktop
  • Keyboard Navigation: Full keyboard support for accessibility

How to Use

  1. Adding Tags: Type in the tags field and press Enter to create new tags
  2. Auto-Complete: Existing tags will appear as suggestions as you type
  3. Multiple Tags: Add several tags to a single expense for better organization
  4. Removing Tags: Click the × button on any tag to remove it
  5. Keyboard Navigation: Use Arrow keys to navigate suggestions, Escape to close
Note: Tags are automatically normalized to lowercase with hyphens (e.g., "Business Lunch" becomes "business-lunch")
Keyboard Shortcuts
  • Enter Add tag
  • Backspace Remove last tag
  • ↑↓ Navigate suggestions
  • Escape Close suggestions
