Upload a CSV file containing restaurant information. The file must include the following required fields:

Required Fields:

The following fields are required in your CSV file:
  • name - Restaurant name
  • address - Street address
  • city - City name
  • state - State/province
  • postal_code - Postal/ZIP code
  • country - Country name

Optional Fields:

  • type - Restaurant type (e.g., restaurant, cafe, bar)
  • price_range - $ to $$$$
  • cuisine - Type of cuisine
  • website - Restaurant website URL
  • phone - Contact number
  • description - Brief description
  • notes - Additional notes
