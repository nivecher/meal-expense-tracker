# Changelog

## 1.0.0 - 2026-03-09

Breaking release that rolls the in-progress workspace into the first stable major version.

### Breaking Changes

- Reworks the merchant and restaurant domain model with new association flows, richer merchant metadata, service dimensions, favicon handling, and updated list and form behavior.
- Introduces loyalty rewards programs, account management, import and export paths, and merchant-to-program linking behind advanced feature access.
- Expands expense workflows with import review, richer detail and list presentations, new clearing data support, and updated service logic for matching and reconciliation.
- Adds admin backup and restore workflows for full user data portability, plus related CLI and remote admin support.
- Includes new migrations, route and template changes, frontend behavior updates, and broader test coverage that require consumers to adopt the latest schema and UI assumptions together.

### Highlights

- Merchant dashboards, summaries, favicon improvements, and import tooling.
- Restaurant search, display preview, association cleanup, and Google Places refinements.
- Expense list, detail, pagination, and offline UX improvements.
- Deployment, workflow, documentation, and developer tooling updates aligned with the new release.
